# 📱 Responsive Web Design & JavaScript To-Do List

This repository contains two front-end web development tasks that focus on creating responsive layouts using **CSS Flexbox & Grid** and building a fully functional **To-Do List Application** with JavaScript and Local Storage.

---

## 📋 Tasks Overview

### 🚀 Task 1: Responsive Web Design with Flexbox & Grid

#### Objectives
- Create responsive web layouts using **CSS Flexbox**.
- Organize content efficiently using **CSS Grid** (e.g., photo gallery or blog layout).
- Ensure the website adapts seamlessly across:
  - 💻 Desktop
  - 📱 Mobile
  - 📟 Tablet

#### Skills Learned
- Flexbox layouts and alignment
- CSS Grid system
- Responsive design principles
- Media Queries
- Modern UI layout techniques

---

### ✅ Task 2: Build a To-Do List App with JavaScript

#### Objectives
- Design the application using **HTML** and **CSS**.
- Implement JavaScript functionality to:
  - ➕ Add tasks
  - ✏️ Update/Edit tasks
  - 🗑️ Delete tasks
- Store tasks using **Browser Local Storage** for persistent data.

#### Features
- Add new tasks
- Edit existing tasks
- Delete completed or unwanted tasks
- Automatically save tasks in Local Storage
- Load saved tasks after page refresh
- Responsive user interface

#### Skills Learned
- JavaScript DOM Manipulation
- Event Handling
- Local Storage API
- CRUD Operations
- Dynamic UI Updates

---
